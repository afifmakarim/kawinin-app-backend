Kawinin Apps - Backend Personal Project
## Getting Started

First, setup environment variable :
```bash
DB_HOST=
DB_USER=
DB_PWD=
DB_NAME=
```

Second, install depency and run the development server
```bash

npm install # install dependency
npm start # start server
```
